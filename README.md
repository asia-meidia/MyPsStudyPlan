## PhotoShop可以做什么？

PhotoShop用于图像处理和图像设计，主要是

* **抠图**

* **修图**

* **调色**

* **合成**

* **特效**


### 1.抠图

在做合成之前，首先要将素材抠好。

如何更快、更好的抠图是每一个设计师的必修课。

### 2.修图

修图可以美化图片，提升图片质量。主要有：

* **修形**

* **修脏**

* **修光影结构**
